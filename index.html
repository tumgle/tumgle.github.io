nvm安装nodejs 环境配置

1、nvm安装nodejs

  • nodejs version manager是nodejs版本管理工具
  • 下载NVM包：地址:https://github.com/coreybutler/nvm-windows/releases, 选择第一个：nvm-noinstall.zip
  • 解压到文件夹，如 D:\Develop\nvm , 里面的文件列表是这样的：elevate.cmd、elevate.vbs、install.cmd、LICENSE、nvm.exe
  • 双击 install.cmd ，输入nvm所在的路径：D:\Develop\nvm 会成一个settings.txt的文本文件，修改内容为

2、nvm环境配置

  • 先删除系统变量中PATH 有关 D:\Develop\nvm和 C:\*\nodejs 的路径
  • 在我们用户变量中添加NVM_HOME和NVM_SYMLINK变量名,然后PATH中添加NVM_HOME和NVM_SYMLINK变量

3、npm环境配置

  • nodejs package manager 也就是nodejs的包管理工具
  • 输入
    npm config set prefix "D:\Develop\nvm\npm"
    然后会在windows用户根目录下生成.npmrc

  • 在添加cache=D:\Develop\nvm\npm-cache

  • ==或者已有npm包，手动配置==，路径还是D:\Develop\nvm\npm，
    手动生成.npmrc文件内容为如上

  • npm环境配置

  • 在Path的最前面添加;%NPM_HOME%

    其他

  • 在项目开始前 npm init 生成package.json 文件会记录项目的依赖，可以npm install 下载所需要的依赖

  • npm install xxx –save-dev

    • 安装一个包到项目本地，必须要联网
    • 安装完成过后项目根目录下会多一个node_modules文件夹，所有的下载下来的包全部在里面
  • npm uninstall xxx –save
    nvm使用
  • nvm use [对应的版本号] [平台架构（如果是32系统需要写32,64位不用管）]
  • nvm install [arch] 安装
  • nvm uninstall 卸载
  • nvm ls 查看已经安装版本
